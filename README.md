everevo Chrome App
==================

everevo（イベレボ）はイベントの集客・集金・参加者管理・コミュニケーションをサポートします。

[Chrome Webstore](https://chrome.google.com/webstore/detail/jhdlcgkifghibofnbnachegncoleaiip)


everevo（イベレボ）とは
-----------------------
ライブ・クラブ・セミナー・イベントの集客管理ツールです。
イベント作成・告知、チケット管理、参加者管理、入場管理などイベントに関するあらゆることをサポートしているサービスです。


主な機能
--------

「かんたんイベント作成！充実のカスタマイズ機能！」

- 無料、有料イベント開催
- イベント告知・参加者募集ページ自動生成
- 告知ページのカラー変更機能
- 参加人数制限機能
- 非公開イベント作成機能
- Facebookページ連携機能

「多彩な告知機能をとりそろえています！」

- 掲示板機能（Facebook連動）
- Facebook上（ウォール）でのイベント情報自動共有・拡散
- Twitter上（タイムライン）でのイベント情報自動共有・拡散
- Facebookフレンドへのイベント招待通知※1回通数制限有り
- Twitterフォロワーへのイベント招待通知※1回通数制限有り
- Gmailアカウントへのイベント招待通知※1回通数制限有り
- 外部サイト（ブログ等）設置用参加ボタン

「手厚い管理機能でイベント主催者をサポート！」

- 複数の管理人設定機能
- 参加者名非表示機能
- 参加者の日本語名管理機能
- 参加者連絡（個別、一括）
- チケッティング（参加チケットメール発行）
- 参加者リスト印刷機能
- 出欠管理機能（来場管理アプリは下をご覧ください）
- リマインダーメール機能
- 1イベント複数チケット発行・購入

「その他、お役立ち機能！」

- Googleカレンダー自動スケジュール登録機能
- イベントのフォロー機能
